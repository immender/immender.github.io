Casse-têtes logiques japonais

Ce site est une archive de casse-têtes logiques publiés par la compagnie japonaise Nikoli.

Qu'est-ce qu'un casse-tête logique ?

Un casse-tête logique est un jeu qui peut être joué seul ou avec l'aide d'autres. Le but est de compléter le casse-tête en suivant les règles du jeu. Un casse-tête logique peut être complété soit avec un papier et un crayon ou de façon numérique. Chaque type de casse-tête suit un ensemble de règles qui lui est propre. La solution peut habituellement être trouvée en utilisant la déduction.

Note

Chaque casse-tête sur ce site web (à moins que l'inverse soit indiqué) inclut un fichier .txt téléchargeable qui peut être importé dans une des applications web sur le site de puzz.link.
